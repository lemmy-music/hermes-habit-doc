# Habit Doc

Flexible habit & event tracking app — create custom widgets for what matters to you.

## Concept

Instead of a fixed tracking catalog, **you design your own tracking cards**:
- **Number fields** (e.g., "glasses of water")
- **Sliders** (e.g., "mood 1-10")
- **Checkboxes** (e.g., "went running")

Log events with **backdated timestamps** for flexibility. Visualize trends and detect **correlations** between habits.

## Tech

- **Flutter** (Web + Mobile, mobile-optimized)
- **SQLite + Drift** (local DB, no cloud)
- **Charts** (timeline & heatmaps)
